# Tender statistics


## Basic Commands

### Migrate

    $ ./manage.py migrate

### Runserver

    $ ./manage.py runserver_plus

### Type checks

Running type checks with mypy:

    $ mypy tender_statistics


### Setting Up Your Users

-   To create a **superuser account**, use this command:

        $ python manage.py createsuperuser
